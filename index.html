FAQ

What is EBA Command Center?

EBA Command Center is a Windows Command-Based Shortcut application. Using EBA Command Center's text box, you can perform a number of features such as executing a file, opening a website, and running commands in Command Prompt.

Im using EBA Cmd 6.1 or below! How do I activate?

Use this activation code for versions 5 to 6.1: 379-EBA-30194-ET

Use this activation code for versions 3.4 to 4.2: 5828-9683-3504

Is EBA Command Center compatible with my device?

EBA Command Center is designed for Windows 10, but can be used for:

  • Windows 10
  • Windows 8.1
  • Windows 8
  • Windows 7

Im using an incompatible device! What can I do?

If you're using an old Windows OS (Windows Vista, Windows XP, Windows ME... etc.), you can upgrade to Windows 7 or Windows 10 (depending on your hardware).

If you're using a MacOS device, you can install VirtualBox, by selecting OS X Hosts. This will download VirtualBox for MacOS. Select the edition of Windows you want to run on your Virtual Machine:

  • Windows 10
  • Windows 8.1
  • Windows 7

If you're using a Linux device, you can install VirtualBox, and select your Linux Distribution (ex: Ubuntu). This will download VirtualBox for your Linux Distribution. Select the edition of Windows you want to run on your Virtual Machine (above).

If your OS wasn't mentioned above, you can try to see if VirtualBox can run on your device. If your using a mobile device (Android, iOS, FireOS... etc), you will need a PC to run EBA Command Center.
