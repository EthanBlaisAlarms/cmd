EBA Command Center Changelog

Upcomming 8.5

  • Improved scripting functions.
  • Plugins
  • Remove secureShutdown and susActivity
  • Changed the URL where the latest version will be downloaded from.
  • Fixxed a bug where plugins would load while installing or updating.
  • Separate app and installer. Users will download the installer, which can install, update, or recover EBA Cmd.
  • The installer will download the EBA Command Center script from the internet.
  • Users can enable auto-update.
  • Updates can be installed by the main script.

EBA 8.41-beta1

  • Plugins are now available to users. Tutorials on plugins coming soon.
  • Improved scripting functions.
  • Plugins can now execute VBS, JS, or EXE files when starting or stopping EBA Command Center. They can also execute those files when an Operation Complete screen pops up, or when a command with the type "script" references it.
  • Plugins can now hold commands within XML files, and localization data in localization.xml

EBA 8.4

  • Fixed a bug that prevented WScript Errors from showing
  • Fixed a bug where EBA Command Center would not connect to the internet on the first try
  • Added plugin support. This is an experimental feature and is not yet fully developed
  • Removed support for EBA Keys. EBA Basic and EBA Professional have been discontinued, all users have a free copy of EBA Enterprise

EBA 8.2

  • Backups are now available to EBA Professional

EBA 8

  • Backups are available for EBA Enterprise, using the 'backup' command
  • Backups can be imported by EBA Basic and above
  • A few minor tweaks to the general UI

EBA 7.31

  • EBA Keys are brought back. EBA Pro and EBA Enterprise are available.

EBA 7.3

  • Support for 32 Bit OS
  • Shutdown Timer can be updated: EBA > Config > Shutdowntimer

EBA 7.21

Minor bug fix with EBA Command Centers check for updates

EBA 7.2

  • EBA Command Center now detects for suspicious activity.
  • EBA Command Center can now read data from this website. This means EBA Command Center can alert you if an update is available.
  • You can also choose to disable your password on your user profile. Simply leave the password text box empty during setup, changing password, or creating a new account.

EBA 7.1

  • EBA File Repair has been replaced with EBA Automatic Repair. You can run EBA Automatic Repair with the 'auto' command in EBA Recovery Mode and EBA StartFail Mode
  • When an issue with the installation occurs during execution, EBA Command Center will crash, with an error message after Operation Completed.
  • When an issue with the installation occurs before EBA Command Center starts, EBA Command Center will go into EBA StartFail Mode, where you can access EBA Recovery and Automatic Repair.
  • Initial setup is now done after EBA Command Center installs, instead of before.
  • When EBA Command Center encounters a Wscript error, you'll get an alert before Operation Completed, instead of auto-crashing EBA Command Center.

EBA 7

  • EBA File Repair is no longer available. A new alternative is being created.
  • EBA Command Center's data is now located at C:\Users\USERNAME\AppData\Roaming\EBA
  • Account and command data is now stored within a single file, instead of within multiple folders.
  • You can now import or export commands using the 'import' or 'export' commands
  • Installs to C:\Program Files (x86)\EBA (updating from 6.1 and below will cause the old version to uninstall)

EBA 6.1

  • EBA File Repair is now active. If EBA Command Center fails to find required files, EBA File Repair might be able to restore it.
  • EBA Recovery Mode is now active. Drop the Recovery Key on the desktop icon to run recovery mode. Commands for recovery mode:
    • Repair
      • File
        • (path) - Run EBA File Repair on the given file.
      • Refresh - Reinstall EBA Command Center
    • Startup
      • (startup) - Change startup type.

EBA 6

  • EBA Command Center is now a single file, meaning the installer is no longer a zip file.
  • Commands can now require login to Administrator to execute.
  • EBA File Repair is available, but is not working.
  • EBA Recovery is in beta stages. Access by dragging the Recovery Key onto the desktop icon.
